# 網頁程式設計fastapi
> ssh private/public key
```bash
ssh-keygen -t ed25519 -C "xx@gmail.com"
```
> Python virtual environment 
```bash
python -m venv venv
.\venv\Scripts\activate
python -m pip install --upgrade pip
```
> FastAPI download with start
```bash
pip install "fastapi[standard]"
fastapi dev main.py
```
